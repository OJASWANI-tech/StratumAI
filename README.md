# 🚀 StratumAI — From Raw Data to Actionable Intelligence

StratumAI is an AI-powered analytics and visualization platform that transforms raw datasets into meaningful insights through intelligent data processing, interactive dashboards, and automated visual analytics.

The platform enables users to upload datasets, explore trends, uncover hidden patterns, detect anomalies, and generate actionable insights through a seamless and intuitive interface.

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/OJASWANI-tech/StratumAI.git
cd StratumAI
```

### Backend Setup

```bash
cd backend
pip install -r requirements.txt
python start_backend.py
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
